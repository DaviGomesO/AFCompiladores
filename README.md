# Compiladores - Atividades

##### Atividade 1 - Foi implementada uma estrutura que recebe um conjunto de caracteres do alfabeto {0,1}, e retorna a quantidade de padrões encontrados para um Automâto Finito Não-Determinístico.

##### Atividade 2 - Foi implementada um Automâto que recebe um conjunto de caracteres do alfabeto que contém todos os digitos, letras, operadores '+' e '-', e o '.', onde o AFND conta padrões de números inteiros e números reais.

##### Atividade 3 (padrSubstring.c) - Foi implementado um algoritmo de força bruta para encontrar um certo padrão determinado dentro de um texto, e adiante um Automâto que venha a encontrar quantas vezes encontra esse padrão no texto, tendo em vista que o automâto melhora a complexidade do algoritmo.

## Translation

##### Activity 1 - A structure was implemented that receives a set of characters from the alphabet {0,1}, and returns the number of patterns found to a Non-Deterministic Finite Automato.

##### Activity 2 - An Automato was implemented that receives a set of characters from the alphabet that contains all the digits, letters, operators '+' and '-', and the '.', where the AFND counts patterns of numbers integers and real numbers.

##### Activity 3 (padrSubstring.c) - A brute-force algorithm was implemented to find a certain pattern determined within a text, and further an Automato that will find how many times it finds this pattern in the text, considering that the automaton improves the complexity of the algorithm.
